Creating Animation & Keyframe

Creating Animation

1. Animation Name

To identify the animation we assign it with the name. In the above example, the name was a demo. So, we will add the attribute animation name.

2. Key Frames Keyframes define

Keyframes define at what % of the animation, and which stylesheet rules should be used. The value of keyframes varies from 0% to 100%. To add keyframes use the % operator with the value inside the keyframes. More keyframes mean smoother animation.
